Checkout the following sample usage of getDeclaredField() and getDeclaredMethod() we can use in unit test.
</Callout>

```java
// Declaring the TestObject's class
Object testObject = new TestObject();

Field field = TestObject.class.getDeclaredField("fieldName");   // Get declared field from the TestObject's class
field.setAccessible(true);                                      // setAccessible to TRUE so that we can access the private field.
field.set(testObject, "fieldValueToBeSet");                     // set our TestObject class's declared field 
                                                                // to the specific value we wanna use for testing.

// Declaring a list of arguments passed to the method.
Object[] arguments = new Object[1];
Argument a = Mockito.mock(Argument.class);

Method method = TestObject.class.getDeclaredMethod("methodName", new Class[]{Argument.class});  // Get declared method from the TestObject's class
method.setAccessible(true);                                                                     // setAccessible to TRUE so that we can access the private method.
method.invoke(testObject, arguments);                                                           // invoke our method in TestObject using our specified arguments in runtime
```
